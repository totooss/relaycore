# RelayCore Roadmap

## 当前仓库内容

当前仓库包含：

- 共享 sessions
- 结构化 command relay
- event timeline 与 digests
- MCP-style memory tools
- Mission Control UI
- token、CORS、audit、redaction、export、backup 相关代码路径
- 本地 memory 迁移能力
